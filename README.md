**How to install**
* Download the ZIP file with the source code and unzip.
* Open "Terminal", enter `cd `, drag in the folder `Uhr` and hit Enter.
* Type `make` and hit Enter to compile Uhr
* `Uhr.app` will be created inside the folder `Uhr`
* Drag `Uhr.app` to the Applications folder (if you want to)

**Compatibilty**
* Down to MacOS 10.11 El Capitan (not tested)
* No Xcode required
* Command Line Tools required, installation is automatically offered when needed
